# SA-MP-0.3.7-Simple-Gamemode
## (ENG):
Easy game mode for SA-MP (0.3.7) which has a system of registration retentive MySQL (plugin v. R39-3). Also used these plugins:

* mysql (pBlueG) R39-3
* streamer 2.7.6
* sscanf2
* dc_cmd 2.8
